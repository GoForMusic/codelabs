Today you will create your first interactive Android application in Java. You will use the Activity class - the major building block of your applications user interface - that defines a single thing the user can do. You will also encounter various widgets to make the UI more interactive, and utilize resources to implement a scalable and maintainable application.

What you'll learn

  • How to create interactive applications using Java
  • How the Activity Lifecycle works
  • How to link XML and Java code
  • How to create Toast and Logcat messages
  • How to use various Android widgets such as a Switch, a Button and a ProgressBar
  • How to externalize and use resources
  • How to localize an application
  • How to provide alternative resources
  • How to work with styles & themes
  • How to work with icons & images
