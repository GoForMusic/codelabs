Today you will learn to create your very first hello world application, focusing on the user interface (UI). The UI that appears on a screen of an Android device consists of a hierarchy of objects called views â€” every element of the screen is a View. The View class represents the basic building block for all UI components. Even layouts such as LinearLayout and ConstraintLayout are subclasses of the View class.
